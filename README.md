## Installation

1. Clone repo
2. Install dependecies
```bash
$ npm install
```
3. Create .env file based on environment variables shown in .env.template
4. Run the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
